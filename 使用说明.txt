# 古风冥想网页 - 使用说明

## 🎯 快速开始

### 1. 准备音频文件

请将以下4个MP3文件放入 `audio` 文件夹中：

| 文件名 | 对应模式 | 推荐内容 |
|--------|---------|---------|
| **focus-guqin.mp3** | 专注冥想 | 古风古琴纯音乐 |
| **relax-rain.mp3** | 放松冥想 | 山林雨声白噪音 |
| **sleep-chant.mp3** | 睡眠冥想 | 轻柔禅音/低频禅意音乐 |
| **gratitude-sing.mp3** | 感恩冥想 | 古风吟唱/佛系轻柔吟唱 |

### 2. 直接打开使用

- **双击** `index.html` 文件
- 浏览器自动打开
- 点击任意冥想卡片（如「🧘 专注冥想」）
- **立即听到音乐！**

## ✨ 功能说明

### 冥想模式
- 🧘 专注冥想：古琴音乐 + 呼吸动画 + 防后台提醒
- 🌸 放松冥想：雨声白噪音 + 全身放松引导
- 🌙 睡眠冥想：禅音 + 夜间模式 + 最后1分钟渐弱
- 🙏 感恩冥想：古风吟唱 + 心情打卡 + 感恩记录

### 操作说明
- **点击冥想卡片** → 立即播放对应音乐
- **点击「开始冥想」** → 进入冥想会话
- **点击暂停按钮** → 暂停/继续播放
- **点击音量按钮** → 调节音量
- **点击返回按钮** → 返回首页

## 🔍 常见问题

**Q: 点击后没有声音？**
A: 请检查 `audio` 文件夹中是否有对应的MP3文件，文件名必须完全一致。

**Q: 浏览器提示无法自动播放？**
A: 已修复！现在点击冥想卡片才会播放，完全符合浏览器规则。

**Q: 手机上能用吗？**
A: 完全可以！手机和电脑都支持，GitHub Pages 部署也兼容。

**Q: 需要服务器吗？**
A: 不需要！双击 index.html 直接打开即可使用！
