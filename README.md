# 每日冥想 - 运动手表连接与智能冥想提醒模块

一个支持运动手表连接的冥想应用，根据用户的运动数据智能推荐冥想内容。

## 功能特性

### 1. 运动手表连接
- 使用 Web Bluetooth API 连接主流运动手表（华为、小米、Garmin 等）
- 支持蓝牙心率、步数、卡路里数据读取
- 自动回退到模拟模式（用于演示和测试）
- 连接状态实时显示：已连接 🟢 / 断开 🔴

### 2. 健康仪表盘
- 实时显示今日步数、卡路里消耗、实时心率
- 每 30 秒自动更新数据
- 卡片式布局，清晰直观

### 3. 运动量不足检测
- 智能检测：步数 < 6000 或 卡路里 < 150 判定为"运动量不足"
- 自动弹出浮动提醒卡片
- 一键进入推荐冥想

### 4. 智能冥想推荐
- 久坐/运动量不足 → 推荐「放松冥想 + 感恩冥想」
- 运动后心率偏高（>110）→ 推荐「专注冥想 + 睡眠冥想」
- 运动量达标 → 随机推荐日常冥想

### 5. 健康数据可视化
- 步数折线图（近7天）
- 卡路里柱状图
- 冥想记录对比展示
- 使用 Canvas 绘制，轻量无依赖

### 6. 打卡系统整合
- 运动量不足触发的冥想自动标记为"健康提醒打卡"
- 记录页面显示健康打卡次数
- 支持生成健康报告卡片并分享

## 技术栈

- HTML5 / CSS3 / JavaScript (ES6+)
- Web Bluetooth API
- Canvas API
- LocalStorage 数据持久化
- 响应式设计，支持移动端

## 使用说明

1. 点击"连接手表"按钮开始蓝牙配对
2. 连接成功后查看健康仪表盘数据
3. 根据今日推荐选择冥想内容
4. 在"我的"页面查看健康数据图表

## 浏览器支持

- Chrome/Edge（支持 Web Bluetooth）
- 其他浏览器自动使用模拟数据模式
